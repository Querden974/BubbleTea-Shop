# Bubble Tea Shop

Bubble Tea Shop est une application web qui permet aux utilisateurs de composer et acheter un bubble tea parmi les produits proposés.

## Technologies

- Vue 3
- Three.js
- CSS + Tailwind + daisyUI
- JavaScript
